## 🚀 发布到 GitHub 的步骤

### 快速发布（如果还没有推送到 GitHub）

在项目根目录打开命令行，执行：

```bash
# 1. 初始化 git（如果需要）
git init

# 2. 添加所有文件
git add .

# 3. 创建首次提交
git commit -m "Initial commit: CSProfHunt - AI-Powered Professor Discovery"

# 4. 添加远程仓库（替换为你的 GitHub 用户名）
git remote add origin https://github.com/Tastal/CSProfHunt.git

# 5. 推送代码
git branch -M main
git push -u origin main
```
